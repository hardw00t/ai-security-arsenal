# System-prompt extraction payloads (2026).
# Frontier models usually refuse direct "show your system prompt" requests.
# Effective modern extraction relies on partial leaks, format-shift tricks,
# and accumulation across turns.

## Direct (baseline — expect refusal on frontier models, but always try first)
